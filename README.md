# 📌 Project Overview

**Rentaly Premium**, gerçek bir araç kiralama firmasının tüm ihtiyaçlarını karşılayabilecek şekilde tasarlanmış modern bir web tabanlı yönetim sistemidir.

Bu proje sayesinde;

✔ Kurumsal panel yönetimi  
✔ Rezervasyon süreçleri  
✔ Araç / Marka / Model yönetimi  
✔ Dashboard analiz ekranları  
✔ Mail bildirim sistemi  
✔ Responsive kullanıcı arayüzü  
✔ Admin operasyon yönetimi  
✔ Gerçek dünya senaryolarına uygun proje deneyimi  

gibi birçok modül başarıyla geliştirilmiştir.

# 🚀 Technologies Used

| Backend | Frontend | Database | Architecture |
|--------|----------|----------|-------------|
| ASP.NET Core MVC | HTML5 | MSSQL Server | N-Tier |
| Entity Framework Core | CSS3 | SQL Queries | SOLID |
| AutoMapper | Bootstrap 5 | LINQ | Repository Pattern |
| Fluent Validation | JavaScript | Stored Procedures | Clean Code |

# 🎯 Main Features

## 🔐 Admin Panel
- Araç ekleme / silme / güncelleme
- Marka & Model yönetimi
- Rezervasyon takip sistemi
- Dashboard yönetimi

## 🚗 Car Management
- Günlük fiyat bilgisi
- Yakıt tipi
- Koltuk sayısı
- KM bilgisi
- Vites türü
- Durum aktif / pasif

## 📊 Dashboard
- Toplam araç sayısı
- Günlük rezervasyonlar
- Marka dağılımı
- Son işlemler
- İstatistik kartları

## 📧 Mail System
- Rezervasyon sonrası mail gönderimi
- Bilgilendirme sistemi
- Kullanıcı iletişim modülü

## 🌐 UI Pages
- Home Page
- Car Listing
- Car Detail
- Reservation
- Contact
- About
- 404 Error Page
